# Student Academic Insights Dashboard

## Project Overview
This project analyzes student academic behavior using data analytics and machine learning techniques. The goal is to understand how factors such as study hours, attendance, and assignment completion influence academic performance. The project also develops a performance scoring system and grade classification model, and presents insights through visualizations and a dashboard.

---

## Dataset Source
The dataset used in this project contains information related to student academic habits and performance, including:

- Study Hours
- Attendance Percentage
- Assignments Completed
- Exam Scores

The dataset was used for educational data analytics practice and was cleaned before analysis.

---

## Project Workflow
The project follows a structured data analysis pipeline:

1. **Data Loading** – Import the dataset into a Jupyter Notebook.
2. **Data Cleaning** – Handle missing values and prepare the dataset.
3. **Exploratory Data Analysis (EDA)** – Analyze patterns using statistical summaries and visualizations.
4. **Feature Engineering** – Create a Performance Score using study hours, attendance, and assignments.
5. **Grade Classification** – Categorize students into grade groups (A, B, C, D).
6. **Machine Learning Model** – Train a classification model to predict student grade categories.
7. **Dashboard Visualization** – Present insights through visualizations and dashboard plots.

---

## Outcomes
The analysis provides insights into student academic performance:

- Students with higher study hours and attendance tend to achieve better grades.
- Assignment completion contributes positively to overall performance.
- The machine learning model predicts grade categories based on academic behavior.
- Visual dashboards help present performance trends and grade distributions clearly.
